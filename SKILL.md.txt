# Install Poetry Skill

## Purpose
Enables agents to set up Python project environments by installing Poetry.

## Usage
Execute the appropriate script for your OS:
- Windows: `scripts/install_poetry.ps1`
- Linux/macOS: `scripts/install_poetry.sh`

## Expected Outcome
- `poetry` command available globally
